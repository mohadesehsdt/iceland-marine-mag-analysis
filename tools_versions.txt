# =====================================================

# General Environment
System: HPC cluster (ROSA, University of Oldenburg)
OS: Linux
Shell: Bash

# Assembly
MEGAHIT v1.2.9

# Binning (via metaWRAP pipeline)
metaWRAP v1.3.2 (conda-based)

  - MetaBAT2 v2.15
  - MaxBin2 v2.2.7

BinChicken (Nature Methods 2023 implementation)

CheckM v1.2.3 (foss-2023a module)

GTDB-Tk v2.4.0

dRep v3.4.2

  - Clustering approach: global dereplication (all MAGs)

Miniconda3 v23.5.2
Python 3.x (via conda environments)

SLURM workload manager
