# Iceland Marine MAG Analysis

## Overview
This project analyzes 171 marine metagenomic samples collected along Icelandic coastal transects. The goal is to reconstruct and analyze metagenome-assembled genomes (MAGs).

## Data
- 171 samples (surface water)
- Sequencing performed at SciLifeLab

## Key Outputs
- Final MAG catalog: `MAG_catalog.tsv`
- Reads per sample: `reads_per_sample.tsv`
- Meta data with reads: `metadata_with_reads.tsv`
